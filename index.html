Programming Competition Resources

This provides resources to aid in preparation of programming competitions. This focuses on ACM International Collegiate Programming Competition (ICPC), solving problems available on the ICPC Live Archive.

The presentations are being made to the UW ACM by Daniel Epstein.

The practice contest run on May 24, 2014 utilized problems from the 2012 ACM Rocky Mountain Regional.

The practice contest run on October 4, 2014 utilized problems from the 2013 ACM South Central Regional.

The UW local qualifier contest run on October 11, 2014 utilized problems from the 2008 ACM Pacific Northwest Regional.

The Pacific Northwest regional contest was on November 15, 2014.

Additional Resources
